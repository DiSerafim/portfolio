Ex. Navegando a partir do parentNode 
parentNode - retorna o elemento pai do nó atual
  • Item 1
  • Item 2

Código:

        <ul id="lista">
            <li>Item 1</li>
            <li>Item 2</li>
        </ul>
        <script>
            // Seleciona o 2º li
            const item2 = document.querySelector('#lista li:nth-child(1)');
            // Retorna id ul
            const pai = item2.parentNode;
            // Cria o novo li
            const novoLi = document.createElement('li');
            // Define o conteúdo do li
            novoLi.textContent = pai.id;
            // Adiciona o novo li
            pai.appendChild(novoLi);
        </script>

Removendo elementos

Feijão Tropeiro

Ingredientes (Removeremos 'toucinho')

Modo de Preparo

  1. Coloque o óleo em uma panela e doure a cebola, acrescente o bacon e frite bem.
  2. Adicione o alho, sal e os ovos, misturando com cuidado para que não se despedacem muito.
  3. Refogue o feijão, baixe o fogo, misture a farinha aos poucos e o cheiro verde.
Código: 
        <h2>Feijão Tropeiro</h2>
        <section class="ingredients-section">
            <h3>Ingredientes (Removeremos 'toucinho')</h3>
            <ul class="ingredient-list">
                <!-- A lista aparecerá aqui -->
            </ul>
        </section>
        <section class="recipe-preparation-section">
            <h3>Modo de Preparo</h3>
            <ol class="recipe-preparation-list">
                <li>Coloque o óleo em uma panela e doure a cebola, acrescente o bacon e frite bem.</li>
                <li>Adicione o alho, sal e os ovos, misturando com cuidado para que não se despedacem muito.</li>
                <li>Refogue o feijão, baixe o fogo, misture a farinha aos poucos e o cheiro verde.</li>
            </ol>
        </section>

        <script>
            let ingredientItem = [
                '500g de feijão carioquinha cozido',
                '200g de toucinho',
                '1 concha de óleo',
                '1 cebola média picada',
                '4 dentes de alho',
                '5 ovos',
                '1 colher de sopa de sal com alho',
                'Cheiro verde a gosto',
                '200g de farinha de mandioca',
            ];

            let ingredientsList = document.querySelector('.ingredient-list');

            for (let i = 0; i < ingredientItem.length; i++) {
                let ingredient = ingredientItem[i];

                let ingredientListItems = document.createElement('li');
                ingredientListItems.innerText = ingredient;
                // Adicionamos uma classe ao li
                ingredientListItems.className = 'ingredients-list-item';

                ingredientsList.appendChild(ingredientListItems);
            };

            let ingredientsListItems = document.querySelectorAll('.ingredients-list-item');

            for (let i = 0; i < ingredientsListItems.length; i++) {
                let element = ingredientsListItems[i];

                if (element.innerText.includes('toucinho')) {
                    ingredientsList.removeChild(element);
                }
            }
        </script>
