Durante minha jornada na Trybe, tive a oportunidade de aprender diversas tecnologias e desenvolver habilidades essenciais para minha carreira. Desde os primeiros passos com Unix, Git e GitHub, compreendi a importância do controle de versão e da organização no desenvolvimento de software.

Aprendi também a estruturar páginas web utilizando HTML semântico e CSS responsivo, garantindo acessibilidade e boa experiência ao usuário. Com JavaScript e ES6, explorei manipulação do DOM, eventos e funções de alta ordem (HOF), fundamentais para criar interações dinâmicas.

Adquirir conhecimento sobre React e Node.js ampliou minha visão sobre desenvolvimento full-stack. SQL foi essencial para entender banco de dados, enquanto empatia, trabalho em equipe e comunicação pelo Slack e Zoom fortaleceram minhas soft skills.

A jornada foi desafiadora, mas enriquecedora. Hoje, me sinto mais preparado para enfrentar desafios na área de tecnologia e seguir evoluindo constantemente.

Lembro-me do momento em que, ao enfrentar dificuldades com testes unitários, precisei buscar novas abordagens para solucionar problemas. Foi um grande aprendizado entender a importância de escrever testes eficientes e garantir a qualidade do código. A persistência me ajudou a superar obstáculos e aprimorar minhas habilidades como desenvolvedor.
